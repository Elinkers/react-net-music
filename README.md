# 文档说明

本项目根据仿照网易云官网进行开发，仅用于学习用途
本项目目前仍在页面开发中，后续会添加功能连接网易云API

## 文件结构树
-react-net-music
|
--config                //webpack设置文档
|   |
|   --jest
|
--public                //root位置
|
--script                //启动配置
|   
--src                   //项目位置
    |
    --component         //公共组件
    |       |
    |       --head      //导航栏
    |
    --home              //首页（内部文件为react组件）


项目启动

### `npm install`
安装依赖包，本项目主要运用react+ant Design

### `npm start`
默认启动端口http://localhost:3000
